# Project creation

Make sure Node and npm are installed
Current Node version is v22.13.1 and npm version is 11.0.0
`npx create-react-app portfolio`

# Tailwind css 

`npm install -D tailwindcss postcss autoprefixer`
`npx tailwindcss-cli@latest init` this generates tailwind.config.js file

# Framer Motion

`npm install framer-motion`

# React icon 

`npm install react-icons`
